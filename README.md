# Neural Network Modeling in Python

## Project Overview

This project demonstrates the implementation of a Neural Network classifier using Python and scikit-learn. The notebook explores how different training iteration values affect model performance by training multiple neural network models and comparing their results.

The project follows a typical machine learning workflow including data preprocessing, feature scaling, model training, and performance evaluation.

## Objectives

- Load and preprocess the dataset.
- Normalize input features.
- Train Neural Network models using different iteration values.
- Compare model performance.
- Visualize the results.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

## Machine Learning Workflow

1. Import libraries
2. Load dataset
3. Clean and prepare data
4. Normalize features using MinMaxScaler
5. Train Neural Network models
6. Compare model performance
7. Visualize results
